# Stellar Compass

Eine digitale Schule für Frauen in Afghanistan.

## Akteure

* **Schülerin**: Lernt, macht Quizze.
* **Mentorin**: Betreut Schülerinnen.
* **Admin**: Verwaltet User und Content.
